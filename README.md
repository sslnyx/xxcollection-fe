# Aimone Frontend

This is the frontend for the Aimone project, built with Vite and React.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm (or yarn) installed.

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/your-username/aimone-fe.git
   ```
2. Install NPM packages
   ```bash
   npm install
   # or yarn install
   ```

### Usage

To run the development server:

```bash
npm run dev
# or yarn dev
```

To build for production:

```bash
npm run build
# or yarn build
```

## Project Structure

- `src/`: Contains the main application source code.
- `public/`: Static assets.
- `components/`: Reusable React components.
- `pages/`: Top-level page components.

## Deployment

This project can be deployed to platforms like Vercel.
